In order to use the program, simply run Main.exe and follow the instructions in the text Prompt.
source code is saved with the exe.

The output will be saved as data.csv.
This is a spreadsheet format, use microsoft word or any other spreadsheet software

                            ---WARNING---
Running the program will override anything previously saved as data.csv.
If you don't want to lose previous data, save the data in another place

                            ---WARNING---
Ensure that the file opencv_world300.dll is packaged with Main.exe
